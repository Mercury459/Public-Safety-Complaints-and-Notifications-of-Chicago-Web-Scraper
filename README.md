# Public-Safety-Complaints-and-Notifications-of-Chicago-Web-Scraper
A web scraper specially designed to extract complaints on misconducts of Chicago police from https://igchicago.org/information-portal/data-dashboards/complaints-notifications-lookup/
Guide: first run Main to get the screeshots of all records, and then run OCR to obtain the records. The results can be used for further analysis like the trend analysis at the end of Main.
